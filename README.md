# TMSLauncher
Taiwai maplestory old version client hook

## Feature
- Fix domain resolve error
- Skip Play Window
- Remove NGS/HS/X3
- Customize game window title
- Multiple game windows

## Test version
- Versions in which HS can't be removed: TMS113/TMS122/TMS145
- Versions in which HS was removed: TMS152/TMS177/TMS183
- Versions in which X3 was removed: TMS189/TMS192
- If you have tested other versions please let me know

## How to use
1. Put Launcher.exe and Hook.dll into MapleStory directory
2. Run Launcher.exe to connect the local game server

## Other
- Change server ip in Launcher.cpp
- Change game window title in Hook.cpp
